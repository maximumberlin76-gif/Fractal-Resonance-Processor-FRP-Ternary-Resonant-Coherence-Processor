# Simulation Layer

This layer contains simulation scenarios and experimental numerical models used to test the Fractal Resonance Processor (FPR) framework.

The simulation layer is focused on:

- phase synchronization behavior
- coherence convergence over time
- resonance-driven acceleration of stable mode formation
- comparison between baseline, resonant, off-resonant, and pulsed scenarios
- dynamic stability under noise and fluctuation

Initial simulations are based on Kuramoto-type phase interaction models and use the order parameter `R(t)` to measure global coherence.

The purpose of this layer is to provide reproducible numerical experiments for testing how selective resonance and external driving influence coherence formation and dynamic stabilization.
